# Pixel Pet 企劃書

## 1. 產品概述

- **產品名稱**：Pixel Buddy Miner (暫定)
- **產品定位**：基於 Nuxt 4 與 Electron 開發的「透明置頂」桌面寵物掛機遊戲。
- **核心賣點**：高度自定義的像素動畫創作、全自動桌面挖礦陪伴、稀有道具收集與社群分享。

## 2. 技術架構

- **前端框架**：Nuxt 4 (Vue 3 Composition API)。
- **桌面環境**：Electron。
- **圖形渲染**：HTML5 Canvas。
- **樣式規範**：手寫 SCSS (遵循 BEM 命名規範) 搭配 CSS 變數。
- **資料結構**：
  - 採用三維陣列儲存：`{ [動作]: [ [ [第1幀像素矩陣] ], [ [第2幀像素矩陣] ] ] }`。
  - 最小儲存單位為像素點的 `HEX` 色值。

## 3. 核心機制 (Core Mechanics)

### A. 自定義像素編輯器

- **畫布規格**：第一版固定為 16x16 像素。
- **動畫模式**：狀態導向幀動畫（使用者需手動繪製每個動作的每一幀）。
- **內建資產**：提供基礎模板（如：史萊姆）及其「走路、站立、休息」動畫，降低初期創作門檻。

### B. 桌面放置與挖礦

- **活動空間**：全螢幕邊緣活動。寵物在螢幕底部（工作列上方）左右自由走動。
- **掛機邏輯**：全自動運行，寵物在活動過程中會隨機產出道具。
- **互動回饋**：
  - 當挖到稀有道具時，寵物會出現「！」或「寶箱」標示並播放驚喜動畫。
  - 玩家點擊寵物後方可收獲道具。
- **道具系統**：第一版僅提供「收集型道具」，用於填滿圖鑑，暫不開放改變數值或畫布規格的功能性道具。

## 4. 使用者介面與體驗 (UI/UX)

- **透明置頂模式**：
  - 支援 Electron 透明背景渲染。
  - 提供「置頂開關」，讓使用者自行決定寵物是否顯示在所有應用程式之上。
- **滑鼠穿透 (Click-through)**：平時寵物不干擾滑鼠點擊操作，僅在需要互動領取獎勵時接收事件。
- **社群互動**：內建截圖功能，方便玩家將挖到的稀有道具與自創寵物分享至社群媒體。

## 5. 跨平台轉化策略

- **Web Demo 版**：
  - 功能：開放像素編輯器與動作展示。
  - 目的：讓使用者體驗「創作寵物」的樂趣，並引導至 Steam 下載完整版。
- **Steam 正式版**：
  - 功能：完整掛機挖礦、道具收集圖鑑、存檔持久化、全螢幕桌面運行。
