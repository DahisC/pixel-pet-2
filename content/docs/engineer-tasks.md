**進度總覽**
- 總任務數：7
- 已完成：7
- 進行中：0
- 待開始：0

**任務清單**

| 任務 | 狀態 | 備註 |
|------|------|------|
| Pinia store + 幀列表面板 | ✅ 完成 | `useEditorStore`、多幀資料結構、幀縮圖顯示、切換幀 |
| 幀操作：新增、複製、刪除 | ✅ 完成 | |
| 動作管理 | ✅ 完成 | 固定 idle/walk/sleep，切換時自動建立第一幀 |
| 即時預覽 + 動畫播放速度 | ✅ 完成 | 重用 `usePixelCanvas`，總秒數控制（0.5s–5s） |
| localStorage 存讀 | ✅ 完成 | 手動讀寫，SPA 模式（ssr: false），頁面載入即還原 |
| 漸層填色工具 | ✅ 完成 | flood fill + 距離轉換，中心偏亮邊緣偏深，快捷鍵 F |
| 首頁顯示玩家創作 | ✅ 完成 | PixelPet 改讀 idle 幀，全空時 fallback 至預設動畫 |

**技術筆記**

- `useEditorStore`（Pinia）管理：動作列表、當前動作、當前幀 index、像素陣列、當前顏色、當前工具
- localStorage key：`pixel-pet-editor`，存整份動作 + 幀資料的 JSON
- SPA 模式（`ssr: false`）確保 localStorage 在 store 初始化時可直接讀取
